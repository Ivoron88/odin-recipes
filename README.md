### Odin Recipes
This project is part of The Odin Project Foundations course. 

### Description
In this project, I am building a basic recipe website to demonstrate my understanding of:
* HTML boilerplate structure
* Working with headings, paragraphs, and lists
* Using relative and absolute links
* Displaying images in HTML
* Git/GitHub workflow (cloning, committing, and pushing)

### Skills Demonstrated
* Clean HTML structure
* Terminal navigation and Git version control
